# Smart Scheduler & Reminder

A modern full-stack MERN application for managing schedules with real-time notifications and alarms.

## Features
- 🔐 **JWT Authentication**: Secure login and registration.
- 📊 **Interactive Dashboard**: Statistics and task overview.
- 📅 **Calendar View**: Monthly view with task markers.
- 🔔 **Reminder System**: Browser notifications and audio alarms.
- 🌓 **Dark Mode**: Seamless theme switching.
- 📱 **Responsive Design**: Optimized for mobile and desktop.

## Prerequisites
- Node.js installed
- MongoDB (local or Atlas)

## Installation

### Backend
1. `cd backend`
2. `npm install`
3. Update `.env` with your `MONGO_URI` and `JWT_SECRET`.
4. `npm run dev`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`

## Tech Stack
- **Frontend**: React.js, Context API, Axios, Lucide React, Framer Motion, Date-fns.
- **Backend**: Node.js, Express, MongoDB, Mongoose, JWT, BcryptJS.
- **Styling**: Vanilla CSS with modern Glassmorphism.
